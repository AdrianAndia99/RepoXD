System Requirements

OS*: Windows 2000/XP/Vista
Processor: 350MHz (minimum); 700+Mhz (recommended)
Memory (RAM): 128MB (minimum); 256+MB (recommended)
Free Hard Drive Space: 50+MB
DirectX Version: 7 (minimum); 9.0c+ (recommended)
Sound: DirectX-compatible
Video: DirectX-compatible; 32MB (minimum)
Color Quality: 16-bit or 32-bit color mode (256 colors may not work)
Controls: Keyboard and mouse
Internet: Internet connection is required to register/unlock game trials - including those delivered via CD-ROM

*You must be an administrator to install this game. If you are using Windows Vista, please ensure that User Account Control (UAC) is activated.

Game Summary

Pieces of an ancient stone tablet have started turning up around the globe and the Museum needs your help to locate the pieces and find the most expensive gem ever known...the "World Diamond". Under the direction of the Museum, you must immediately begin your global seek & find adventure.

Complete over 100 levels by finding hidden objects and circumventing unique puzzle traps.

Troubleshooting

This software has been tested to operate on the above system requirements. If you experience any problems such as system crashes, any graphics issues or error messages, please try one of the following:

1) Update your computer's video or sound card drivers. This can be done by identifying the type of sound card and video card your computer is using and finding the manufacturer's web site where new drivers may be made available for download.

2) Upgrade DirectX. This is a Microsoft component and can improve the graphics, security, and performance of your Windows operating system.

3) If 1 and 2 do not fix graphic problems please go to the "Options" menu available on the main menu and UNCHECK "Use Hardware Acceleration". If you change the "Use Hardware Acceleration" you need to close the game and launch it again for the new setting to begin.

4) If you are having general issues with the game or your PC is running slow it is always a good idea to restart your PC.
