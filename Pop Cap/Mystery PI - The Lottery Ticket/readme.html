Minimum System Requirements

-------------------

* Windows XP/2000. If you are using Windows Vista, please ensure that User Account Control (UAC) is activated. You must be logged in as an administrator to install this game.

* 350Mhz Pentium or faster CPU

* 128 MB RAM or higher

* 16-bit Graphics and Sound card

* DirectX7 or later

* 50 MB free hard drive space

Game Summary

---------------

You are the world famous Mystery P.I. and you've been hired by Grandma Rose to find her winning $488 Million lottery ticket! Retrace Grandma's footsteps to collect clues by finding over 2100 cleverly hidden objects and solving puzzles. If you find the ticket before the deadline you will be awarded $20 Million.

As your investigation skills improve your P.I. rank increases -- strive for a "Perfect Investigation" to get big bonus points. Two addictive ways to play; Find the Ticket and Unlimited Seek & Find game mode. Find the Clues, Solve the Mystery!

Search for the hidden keys to unlock Unlimited Seek & Find mode and a cool bonus level. Good investigating!

Troubleshooting

---------------

This software has been tested to operate on the above system requirements. If you experience any problems such as system crashes, any graphics issues or error messages, please try one of the following:

1) Update your computer's video or sound card drivers. This can be done by identifying the type of sound card and video card your computer is using and finding the manufacturer's web site where new drivers may be made available for download.

2) Upgrade DirectX. This is a Microsoft component and can improve the graphics, security, and performance of your Windows operating system.

3) If 1 and 2 do not fix graphic problems please go to the "Options" menu available on the main menu and UNCHECK "Use Hardware Acceleration". If you change the "Use Hardware Acceleration" you need to close the game and launch it again for the new setting to begin.
